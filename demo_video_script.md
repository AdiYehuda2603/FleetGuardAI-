## 🎥 הוראות הקלטה - צעד אחר צעד

### אפשרות 1: OBS Studio (מקצועי ביותר - מומלץ!)

#### התקנה:
1. הורד מ: https://obsproject.com/download
2. התקן (Next → Next → Finish)
3. פתח OBS Studio

#### הגדרות:
1. **Scene Setup:**
   - לחץ ➕ תחת "Sources"
   - בחר "Display Capture" (לכידת מסך שלם)
   - או "Window Capture" (רק חלון Streamlit)

2. **Audio:**
   - Settings → Audio
   - Desktop Audio: Default (לצלילי מערכת)
   - Mic/Auxiliary: בחר מיקרופון (אם רוצה קריינות)

3. **Video Settings:**
   - Settings → Video
   - Base Resolution: 1920×1080
   - Output Resolution: 1920×1080
   - FPS: 30

4. **Recording Settings:**
   - Settings → Output
   - Recording Quality: High Quality
   - Recording Format: MP4
   - Encoder: x264

#### הקלטה:
1. לחץ **Start Recording** (או F11)
2. בצע את התסריט (5 דקות)
3. לחץ **Stop Recording** (או F11)
4. הקובץ יישמר ב: `C:\Users\[שם משתמש]\Videos\`

---

### אפשרות 2: Windows Game Bar (מהיר ופשוט)

#### הפעלה:
1. לחץ **Win + G** (מקלדת)
2. יפתח חלון Game Bar

#### הקלטה:
1. לחץ על כפתור ההקלטה ⚫ (או Win + Alt + R)
2. בצע את התסריט
3. לחץ ⏹ Stop (או Win + Alt + R שוב)
4. הקובץ נשמר ב: `C:\Users\[שם]\Videos\Captures\`

**חסרונות:**
- לא יכול להקליט Desktop (רק חלונות אפליקציות)
- פחות אפשרויות עריכה

---

### אפשרות 3: Loom (קל מאוד, אבל מוגבל ל-5 דקות בחינמי)

#### התקנה:
1. הורד מ: https://www.loom.com/download
2. התקן והתחבר (אימייל חינמי)

#### הקלטה:
1. פתח Loom
2. בחר "Screen + Camera" או "Screen Only"
3. בחר את המסך/חלון להקלטה
4. לחץ **Start Recording**
5. בצע תסריט (עד 5 דקות!)
6. לחץ **Stop**
7. הסרטון מועלה אוטומטית ל-Cloud
8. הורד כ-MP4

**יתרון:** קל מאוד, עריכה בסיסית מובנית
**חסרון:** מוגבל ל-5 דקות (מתאים בדיוק!)
