# ✅ תיקון שמות עמודים - FleetGuard

## הבעיה:
Streamlit לא הצליח למצוא עמודים עם אימוג'ים בשם הקובץ:
- `1_🔐_כניסה.py` ❌
- `2_📝_הרשמה.py` ❌

## הפתרון:
שינוי שמות הקבצים לשמות פשוטים יותר:
- `1_Login.py` ✅
- `2_Register.py` ✅

## מה שונה:

### קבצים חדשים:
1. **`pages/1_Login.py`** - עמוד כניסה
2. **`pages/2_Register.py`** - עמוד הרשמה

### עדכונים:
- `main.py` - עודכן להשתמש ב-`1_Login`
- כל ה-`st.switch_page()` עודכנו לשמות החדשים

## שימוש:
```python
# מעמוד ראשי לעמוד כניסה
st.switch_page("1_Login")

# מעמוד כניסה להרשמה
st.switch_page("2_Register")

# מעמוד הרשמה לכניסה
st.switch_page("1_Login")

# מעמוד כלשהו לעמוד ראשי
st.switch_page("main")
```

## הערות:
- שמות הקבצים החדשים ללא אימוג'ים
- כל הפונקציונליות נשמרה
- האימוג'ים עדיין מוצגים בכותרת העמוד
